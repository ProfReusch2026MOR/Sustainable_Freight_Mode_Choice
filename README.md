# Sustainable Freight Mode Choice

Plan shipments through a multimodal transport network with road, rail, and transfer terminals. The project should decide which routes and modes to use for shipments while considering cost, emissions, delivery times, terminal handling, and capacity.

Minimum complexity: include multiple shipments, different transport modes, transfer costs, delivery deadlines, capacity limits, and a cost-emission trade-off. The large instance should include many shipments and a dense network where consolidation decisions matter.

Data requirements: nodes, road arcs, rail arcs, transfer terminals, shipment origins and destinations, shipment volumes, transport costs, travel times, emissions, handling costs, capacities, and deadlines.
